Cómo ejecutar el proyecto en Dev-C++ (paso a paso)

1. Crear una carpeta vacía
- Ir al escritorio y crear una carpeta nueva.
- Por ejemplo: ProyectoC

2. Clonar el repositorio dentro de esa carpeta
- Abrir CMD (Símbolo del sistema).
- Navegar hasta la carpeta creada, por ejemplo: cd Desktop\ProyectoC

- Ejecutar: git clone https://github.com/joseviudes/8escalones-aed2
* Esto descargará todos los archivos .c y .h dentro de la carpeta.

3. Abrir Dev-C++ y crear un proyecto nuevo
- Abrir Dev-C++
- Ir a Archivo → Nuevo → Proyecto
- Elegir “Proyecto vacío (Empty Project)”
- Guardarlo con un nombre, por ejemplo: TrabajoFinalC

4. Agregar los archivos del repositorio al proyecto
- En Dev-C++, en el panel izquierdo, hacer clic derecho sobre el proyecto.
- Seleccionar “Añadir al proyecto…”
- Buscar la carpeta clonada (la que tiene los .c y .h).
- Seleccionar todos los archivos .c y .h y agregarlos.

5. Seleccionar main.c y ejecutar
- En el panel del proyecto, abrir main.c.
- Presionar F11 o el botón “Compilar y ejecutar”.
- El programa comenzará a funcionar.
